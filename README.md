# Vending Machine

![Vending Machine Frame](./img/2%20Frame.JPG)

## Equipment

- Laser Cutter/CNC Router (I used a glowforge)
- Drill
- 3/32" Wood Drill Bit
- Small screwdriver (for M2.5 screws)
- Wire Cutters
- Short piece of 2" PVC (about 16"-24" long) to create the spirals
